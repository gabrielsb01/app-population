# Documentação do Projeto

Neste documento, descreveremos a estrutura e as tecnologias utilizadas no projeto de visualização da evolução populacional dos bairros.

## Tecnologias Utilizadas

- **Vite.js**: Utilizamos o Vite.js como nossa ferramenta de build para desenvolvimento rápido e eficiente com React e TypeScript.
- **React**: Escolhemos React como nossa biblioteca principal para desenvolver interfaces de usuário dinâmicas e reativas.
- **TypeScript**: Utilizamos TypeScript para adicionar tipagem estática ao JavaScript, garantindo maior robustez e prevenindo erros.
- **Storybook**: Utilizamos o Storybook para documentar e visualizar os componentes do nosso projeto, permitindo um desenvolvimento mais rápido e testes visuais.
- **react-chartjs-2**: Para a exibição dos gráficos de evolução populacional, utilizamos a biblioteca react-chartjs-2, que oferece uma integração fácil e flexível com o Chart.js.
- **react-leaflet**: Utilizamos o react-leaflet para renderizar o mapa e visualizar os bairros, proporcionando uma experiência interativa ao usuário.
- **Cypress**: Para garantir a qualidade do nosso projeto, implementamos testes end-to-end automatizados com o Cypress, permitindo uma validação abrangente da aplicação por meio de simulações de interações do usuário.
